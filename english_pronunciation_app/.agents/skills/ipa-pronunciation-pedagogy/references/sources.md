# Sources For IPA Pronunciation Pedagogy

Use these sources as reference anchors. Do not copy copyrighted lesson text or audio into the project.

## Primary IPA Reference

- International Phonetic Association, IPA Chart: https://www.internationalphoneticassociation.org/content/ipa-chart
  - Use for official IPA symbol checking and broad phonetic categories.

## Teaching Pronunciation

- British Council TeachingEnglish, pronunciation and phonemic chart resources: https://www.teachingenglish.org.uk/
  - Use for classroom-oriented sequencing ideas and phonemic awareness activities.

- Cambridge English teaching resources: https://www.cambridgeenglish.org/
  - Use for general English-learning and exam-oriented teaching guidance.

## Project Policy

- Treat IPA/audio as content requiring human review.
- Do not copy full exercises, book content, dictionary audio, or proprietary examples.
- For MVP, prefer project-authored examples and local audio created by the project team.
