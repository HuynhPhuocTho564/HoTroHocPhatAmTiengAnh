# Sources For Question Bank Curation

Use these sources to guide structure, review, and licensing decisions. Verify current terms before using any external data in the app.

## Assessment Item Structure

- 1EdTech Question and Test Interoperability (QTI): https://www.1edtech.org/standards/qti
  - Use as a conceptual reference for separating prompts, interactions, response data, scoring, and metadata.

- National Board of Medical Examiners, item-writing guidance: https://www.nbme.org/
  - Use for general multiple-choice quality principles: clear stem, plausible options, one best answer, avoid trick wording.

## IPA And Pronunciation Sources

- International Phonetic Association, IPA Chart: https://www.internationalphoneticassociation.org/content/ipa-chart
  - Use for official IPA symbol checking.

- British Council TeachingEnglish: https://www.teachingenglish.org.uk/
  - Use for pronunciation teaching patterns and minimal-pair activity ideas.

## Dictionary And Audio Licensing Caution

- Cambridge Dictionary: https://dictionary.cambridge.org/
  - Use only as manual reference unless the project has explicit permission or API/license terms allowing storage and app usage.

- Oxford Learner's Dictionaries: https://www.oxfordlearnersdictionaries.com/
  - Use only as manual reference unless the project has explicit permission or API/license terms allowing storage and app usage.

- Creative Commons license chooser and legal references: https://creativecommons.org/
  - Use to check attribution, noncommercial, share-alike, and derivative requirements before importing open content.

## Project Policy

- Do not scrape or bulk-download dictionary audio.
- Do not seed `ACTIVE` content from an external source unless source rights and review status are clear.
- Store source metadata even for manually authored data when it helps later review.
